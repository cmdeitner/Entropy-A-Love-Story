## About the Collection

This is an assignment for graduate school teaching us how to begin to use CollectionBuilder-GitHub. I hope to learn how: 

- To use Markdown
- To use the GitHub web interface.

## Collection History

This is a collection of artifacts connected to a the play "Entropy, A Love Story", written, directed and produced by Christine Deitner. This production occurred in Los Angeles, in 2012. The script had previously been produced as a Workshop in Northampton, MA.

The template repository features four objects from the University of Idaho Library's [Digital Collections](https://www.lib.uidaho.edu/digital).
